# ADR-004: 支援者 UX サルベージ

**日付**: 2026-07-19  
**状態**: 採用

## 決定

1. **SOS 状況プリセット** — `sos-templates.ts` で被災者入力を復元（既存自由入力は維持）
2. **位置 UI** — 現在地・微調整は 🚨 タブ内のみ。微調整時は検索タブへ遷移しない
3. **contact_info** — 認証済み RPC（`0002`）のみ返却。地図・公開型からは除外したまま
4. **成功トースト削除** — 位置取得は地図ピンで十分。右下通知はエラー時のみ
5. **Bootstrap 増分適用** — 通常は `0002+` のみ（`0001` §0 RESET でデータ消去しない）

## 根拠

README「投稿は開、閲覧は閉」と矛盾しない範囲で、支援者が連絡先・詳細を一覧で確認できるようにする。
